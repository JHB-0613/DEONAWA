저장장치(SSD/HDD)

보조 기억장치

운영체제, 프로그램, 문서, 사진, 음악, 영상 같은 모든 데이터를 장기적으로 저장하는 컴퓨터 부품이다.

NVMe SSD(M.2)/SATA SSD/HDD 등. 웬만하면 속도가 빠른 SSD를 우선적으로 장착한다.

역할 및 필요성

데이터 영구 저장 공간 제공

컴퓨터가 실행하는 모든 프로그램, 운영체제, 개인파일(문서, 사진, 영상 등)은 저장장치에 저장됨.

전원을 꺼도 데이터가 사라지지 않고 계속 유지되기에, '기억장치' 역할을 하는 필수 부품.

운영체제 부팅과 프로그램 실행

운영체제(윈도우, 맥OS 등)와 프로그램은 저장장치에 설치되어 있어야 부팅과 실행이 가능.

저장장치가 없으면 컴퓨터는 아무것도 불러올 수 없어 작동하지 않음.

데이터 읽기/쓰기의 근본 수단

프로그램 실행 시 필요한 파일을 저장장치에서 불러와 RAM으로 옮기고, 결과물을 다시 저장장치에 저장.

즉, 컴퓨터의 기본 동작 사이클에서 필수적인 역할을 함.

저장장치가 없으면?

컴퓨터가 운영체제를 불러올 수 없어 부팅 자체가 불가능.

RAM은 휘발성이기 때문에 저장장치 없이 작업 데이터를 유지할 수 없음.

프로그램 설치, 파일 저장, 업데이트 모두 불가능.
